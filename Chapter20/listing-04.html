div {
                width: 75%;
                height: 100px;
                border: thin solid black;
            }
            img {
                background: lightgray;
                border: 4px solid black;
                margin: 2px;
                height: 50%;
            }
            #first {
                box-sizing: border-box;
                width: 50%;
            }
            #second {
                box-sizing: content-box;
            }
    
 widthを%で指定すると親ボックスの大きさに合わせて伸張する。
 ２つ目のイメージではheightのみを指定しているので、それに合わせてwidthを計算している。
 
 box-sizingでwidthとheightの指す領域を指定することができる
 ２つのイメージはいれずれもheightが50%だが高さが異なるのはfirstが高さにborderを含んでいるため
